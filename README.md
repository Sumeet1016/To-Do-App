# React_Projects - To-Do App

This is a **To-Do App** built with **React** and **Vite**.  

### Features
- Add tasks with a title and due date
- Mark tasks as completed
- Delete tasks
- Styled with CSS Modules and Bootstrap

### Tech Stack
- React
- Vite
- CSS Modules
- Bootstrap
